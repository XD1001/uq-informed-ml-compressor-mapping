# Data Files

This folder contains cleaned CSV exports for the three case-study datasets used by the example scripts.

## Files

- `case_1_rotary.csv`: constant-speed R410A dual-cylinder rolling-piston compressor.
- `case_2_scroll.csv`: constant-speed R454C scroll compressor.
- `case_3_vi.csv`: variable-speed R410A vapor-injection scroll compressor.

The CSV files were assembled from previously published and publicly accessible literature sources cited in the associated article.
